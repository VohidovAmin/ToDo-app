# Todo App — Управляй задачами просто

Простое и удобное приложение для ведения списка задач.  
Сделано на чистом HTML, CSS и JavaScript. Подходит для ежедневного использования.

## 🚀 Возможности

- ➕ Добавление новых задач (кнопка или Enter)
- ✅ Отметка выполненных задач (красивые чекбоксы с галочкой)
- 🗑️ Удаление задач с плавной анимацией
- 🔍 Фильтры: Все / Активные / Выполненные
- 📊 Счётчик оставшихся задач
- 🌙 Тёмная и светлая тема
- 📱 Адаптивный дизайн — удобно на телефоне и компьютере
- 💾 Автоматическое сохранение задач в браузере

## 🛠️ Технологии

- HTML5
- CSS3 (Flexbox, переменные, адаптивность)
- JavaScript (ES6+)
- LocalStorage API
